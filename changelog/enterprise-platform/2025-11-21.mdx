## What's New

- **Mobile route creation**: Create patrol routes directly from your mobile device using GPS - simply walk the route and tap to add checkpoints
- **Single checkpoint routes**: Routes can now be created with just one checkpoint for simpler patrol scenarios
- **Assignment priority overrides**: Set priority levels for assignments with the ability to override when needed
- **Smart login redirects**: When signing in, you're now returned to the page you were trying to access

## Improvements

- **Better GPS accuracy**: Improved location accuracy for mobile patrol tracking with adaptive timing based on signal quality
- **Checkpoint configuration**: Enhanced checkpoint settings including radius, speed limits, terrain type, and custom instructions
- **Improved people creation**: Streamlined wizard for adding new people with better organization and validation
- **Roster management refinements**: Better calendar controls and filtering options for roster management

## Bug Fixes

- Fixed issue where admin menu wasn't visible for some users
- Fixed user data display issues after signing in
- Resolved various display and interaction issues across dashboard components
