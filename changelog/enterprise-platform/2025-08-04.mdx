## What's New

- **Custom dashboards**: Create personalized dashboards with drag-and-drop widgets
- **Card widgets**: Add data cards showing key metrics at a glance
- **Bar charts**: Visualize data with bar charts and grouping options
- **Line charts**: Track trends over time with line chart widgets
- **Widget management**: Full create, edit, and delete functionality for all widgets
- **Drag to reorder**: Easily rearrange dashboard widgets by dragging

## Improvements

- **System skills**: System skills now display separately and auto-add to new groups
- **Admin skill config**: New admin configuration for managing system skills
- **Alert flows**: Connected field selection now available in alert flow builder
