## What's New

- **Patrol Events Dashboard**: Complete overview of your patrol operations including active assignments, event feeds, and performance charts showing events over time and top patrol assignments
- **Access Control Dashboard**: At-a-glance view of vehicle and person access activity with analytics on entry/exit distribution, top vehicle makes, and staff scan activity
- **Roster Management Dashboard**: Comprehensive scheduling overview with stats on active schedules, hours worked, coverage gaps, and visual breakdowns of schedule types
- **Area-based map navigation**: Map now auto-zooms to your configured areas and shows helpful prompts when areas need to be set up
- **Enhanced export builder**: Select up to 8 fields (increased from 6) with example values shown for each field to help you build the perfect export

## Improvements

- **Unified People & Roster exports**: People and roster data are now combined into a single collection for easier export configuration
- **Improved schedule wizard**: Schedule creation interface is now more intuitive with better visual consistency
- **Better mobile popups**: Cleaner, more readable popup designs on mobile devices
- **Feature-based access control**: Business tools now only display for users with appropriate permissions

## Bug Fixes

- Fixed checkpoint rate always showing 100%
- Fixed average distance calculation for patrol routes
