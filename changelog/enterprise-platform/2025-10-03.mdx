## What's New

- **Multi-filter support**: Apply multiple filters to dashboard widgets for more precise data views
- **Spacer widgets**: Add visual spacers to organize your dashboard layout
- **Relative date ranges**: Filter by rolling time windows (last X hours/days/weeks) or calendar periods
- **Improved widget configuration**: Widgets now open configuration dialogs immediately when added

## Improvements

- **Better schedule form design**: Cleaner step-by-step schedule creation
- **Enhanced widget cards**: More information displayed on dashboard widget cards
- **Group ID filtering**: Filter widgets by specific groups

## Bug Fixes

- Fixed filter dropdowns being incorrectly disabled when adding multiple filters
- Fixed widget configuration not loading when editing existing widgets
- Fixed data disappearing after saving widget configuration
