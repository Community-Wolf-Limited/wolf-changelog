## What's New

- **WhatsApp Integration**: Full conversational AI agent accessible via WhatsApp for incident reporting and area monitoring
- **Interactive Map Dashboard**: Real-time incident visualization with Mapbox GL JS, including map markers, filtering, and location tracking
- **AI-Powered Conversations**: Chat history and context-aware responses using Groq/OpenAI for natural language incident reporting
- **Weather Integration**: Real-time weather data display for situational awareness
- **User Authentication**: Secure login system with NextAuth.js
- **Geolocation Support**: GPS-based location detection for accurate incident positioning

## Technical Foundation

This release establishes the core architecture:
- Next.js App Router with API routes
- MongoDB for data persistence
- Twilio for WhatsApp messaging
- Mapbox GL JS for mapping
- OpenAI/Groq for AI processing
