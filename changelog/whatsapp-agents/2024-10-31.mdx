## What's New

- **WhatsApp Message Templates**: Pre-approved message templates for proactive notifications and alerts
- **AWS Media Storage**: Images and videos now stored securely in AWS S3 for reliable media handling
- **Enhanced Authentication**: Email/password authentication with improved security
- **Area Selection**: Users can now select and manage monitoring areas directly from the map
- **GeoJSON Support**: Import and display custom geographic boundaries

## Improvements

- **Map Visualization**: Updated polygon colors and improved visual hierarchy for area boundaries
- **Number Validation**: Phone number validation for WhatsApp registration
- **Chat History Timing**: Optimized chat history retrieval for faster response times
- **Image Processing**: Enhanced AI prompts for better image analysis and description
