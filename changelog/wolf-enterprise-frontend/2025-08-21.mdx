## What's New

- **Webhook integration**: Connect alert flows to external services via webhooks with bearer token authentication
- **Trigger flexibility**: Alert flows can now trigger on any matching event
- **Feature access controls**: Granular permission controls with visual indicators for restricted features
- **Blurred preview**: Restricted features show a blurred preview with upgrade prompts

## Improvements

- **Better group agent filtering**: Improved filtering options for agent groups
- **Alert flow builder fixes**: More reliable alert flow configuration
