## What's New

- **Report Intelligence dashboard**: Visualise report networks with interactive graphs showing connections and relationships.
- **Intelligence map view**: Explore report locations and patterns on an interactive map with filtering and focus controls.
- **Alert metrics**: See alert delivery counts and performance metrics directly in report views.
- **Patrol event comments**: Add and view comments on patrol events for better documentation and team communication.

## Improvements

- **Network exploration**: Easily navigate report relationships with graph and map views that highlight important connections.
- **Filtering and focus**: Quickly focus on specific reports or areas of interest in intelligence views.
- **Comment badges**: Visual indicators show which patrol events have comments for quick reference.

## Bug Fixes

- **Report sorting**: Fixed issue where user sort preferences weren't respected in merged report views.
- **Real-time updates**: Improved consistency of report sorting and filtering in live updates.
