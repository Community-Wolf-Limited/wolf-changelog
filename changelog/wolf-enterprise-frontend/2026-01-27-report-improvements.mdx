## Improvements

- **Sort preference respect**: User sort preferences are now consistently applied across all report views.
- **Filter persistence**: Report filters maintain their state during real-time updates.

## Bug Fixes

- **Merged report sorting**: Fixed issue where merged reports weren't using the user's preferred sort order.
- **Socket update consistency**: Real-time report updates now respect user sort preferences and active filters.
- **Source type sorting**: Improved sorting functionality to handle different report source types correctly.
