## What's New

- **Patrol event detail modals**: View comprehensive patrol event details in an expanded modal view
- **Interactive site management**: Better integration between sites and patrol checkpoints on maps
- **Route calculations**: Automatic calculation of distances and estimated times between checkpoints

## Improvements

- **Better map precision**: Fixed coordinate offset issues for more accurate checkpoint placement
- **Enhanced error handling**: Better error boundaries prevent crashes from affecting other features
- **Improved permission handling**: More consistent access control across patrol features
- **Responsive design**: Better layout on mobile devices with improved save buttons and map settings
