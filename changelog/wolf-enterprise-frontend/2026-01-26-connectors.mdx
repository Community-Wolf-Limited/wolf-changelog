## What's New

- **Connector builder**: Create and configure email connectors with a user-friendly interface.
- **Email template presets**: Get started quickly with pre-configured email templates.
- **Test email dialog**: Verify connector configurations by sending test emails before activating.
- **Organisation placeholders**: Use dynamic placeholders in email templates for personalised messaging.

## Improvements

- **Input validation**: Enhanced validation and error handling in connector configuration forms.
- **Theme persistence**: System theme preference now persists across sessions for better user experience.
- **Feature access control**: Connector pages are properly protected based on feature access permissions.

## Bug Fixes

- **Stale user data**: Fixed issue where connector user lists could show outdated information after organisation changes.
