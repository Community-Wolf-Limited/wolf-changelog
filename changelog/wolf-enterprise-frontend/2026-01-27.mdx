## Improvements

- **Default report view**: Full report set by default instead of auto-limiting to organisation areas unless you choose that filter.
- **Verified filter**: Applies only when explicitly enabled.
- **Connector testing**: Test flows include users linked through staff records.
- Release integration: `general-fixes`.

## Bug Fixes

- **Filtering**: Area filters no longer hide reports when users expect the full list.
- **Sorting**: More consistent ordering across views and live updates.
